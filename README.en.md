# Esports Talent Test

[中文](README.md) | [English](README.en.md)

A Douyin virtual interactive-space project. It uses six lightweight challenges to test reaction speed, tapping speed, peripheral vision, tracking, memory, and hand-eye coordination. Scan and play instantly to see who has esports talent.

Live demo: [https://swording-k.github.io/esports-talent/](https://swording-k.github.io/esports-talent/)

## Highlights

- Six-dimensional esports assessment: reaction, tapping speed, vision, tracking, sequence memory, and coordination
- Mobile-first experience for Douyin’s in-app browser and portrait phone screens
- Game-like visual design with a dark esports theme, orange accent color, HUD interface, and instant feedback animations
- Local best-score records stored with `localStorage`

## Challenges

| Test | What It Does | Ability |
| --- | --- | --- |
| Reaction Speed | Click as soon as the signal turns green | Reaction time |
| Tapping Speed | Tap targets as many times as possible in 30 seconds | Click speed |
| Peripheral Vision | Memorize colored dots in a short time | Observation and memory |
| Dynamic Tracking | Keep tracking a moving target | Target tracking |
| Sequence Memory | Repeat the highlighted arrow sequence | Short-term memory |
| Hand-Eye Coordination | Catch falling targets with a slider | Coordination |

## Run Locally

Open `index.html` directly, or start a local static server:

```bash
python3 -m http.server 8000
```
