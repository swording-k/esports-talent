# 电竞天赋检测站

[中文](README.md) | [English](README.en.md)

抖音虚拟平台互动空间作品：6 项趣味挑战测试反应、手速、视野、追踪、记忆、协调能力。扫码即玩，30 秒完成单项挑战，看看谁是电竞天才。

在线体验：[https://swording-k.github.io/esports-talent/](https://swording-k.github.io/esports-talent/)

## 功能亮点

- 六维电竞能力测评：反应速度、极限手速、视野广度、动态追踪、顺序记忆、手眼协调
- 移动端优先体验：适合抖音内置浏览器和手机竖屏访问
- 游戏化视觉：黑色电竞背景、橙色主题色、HUD 风格界面和动效反馈
- 本地最佳成绩：使用 localStorage 记录各项历史最好成绩

## 测试项目

| 测试 | 内容 | 目标能力 |
| --- | --- | --- |
| 反应速度 | 等待信号变绿后立刻点击 | 反应时间 |
| 极限手速 | 30 秒内尽可能多地点击目标 | 点击速度 |
| 视野广度 | 短时间记忆彩色圆点 | 观察与记忆 |
| 动态追踪 | 持续追踪移动目标 | 目标追踪 |
| 顺序记忆 | 复现箭头亮起顺序 | 短时记忆 |
| 手眼协调 | 用滑块接住下落目标 | 操作协调 |

## 运行方式

直接打开 `index.html` 即可体验，也可以启动本地静态服务器：

```bash
python3 -m http.server 8000
```
